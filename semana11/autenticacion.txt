El login es el proceso mediante el cual un usuario ingresa a una aplicación utilizando sus credenciales para verificar su identidad. Una vez autenticado, puede acceder a las funciones permitidas según su perfil. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar acciones importantes, como crear, modificar o eliminar datos. De esta forma se evita el acceso no autorizado. No deben exponerse públicamente contraseñas, documentos de identidad, direcciones, teléfonos, tokens, claves de API ni otros datos sensibles de los usuarios o de la comunidad.
